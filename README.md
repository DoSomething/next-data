## next-data

This repo is a dashboard for all of the Keen.Io data we collect on Phoenix Next.

## Setup

```sh
$ npm install
$ cp .env.example .env
```

## Standalone app

```sh
$ node app.js
```

## Slack Bot

```sh
$ node bot.js
```

## Usage

```
${process.env.MAGIC_PHRASE} [campaignId||null]
```
